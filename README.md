# 📊 Dashboard de Salários na Área de Dados

## 🌐 Acesse o Dashboard

https://dashboard-salarios-dados-rhxf6mzntfyugj4q4eogxd.streamlit.app/


Este projeto é um dashboard interativo desenvolvido em **Python** utilizando **Streamlit** para análise de salários na área de dados.

O dashboard permite explorar informações sobre **cargos, níveis de senioridade, tipo de contrato, tamanho das empresas e distribuição geográfica** dos profissionais da área de dados.

## 🚀 Tecnologias utilizadas

- Python  
- Pandas  
- Plotly  
- Streamlit  

## 📂 Fonte dos dados

Os dados utilizados neste projeto estão disponíveis publicamente no GitHub.

📊 Insights obtidos

Os maiores salários da área de dados estão concentrados nos Estados Unidos, indicando maior maturidade do mercado de tecnologia e dados nesse país.

Existe uma diferença salarial significativa entre países desenvolvidos e emergentes, evidenciando desigualdade global na remuneração da área de dados.

O cargo Data Scientist aparece como o mais frequente no dataset, sugerindo forte demanda por profissionais com habilidades em análise e modelagem de dados.

Profissionais com maior nível de senioridade apresentam salários significativamente mais altos, mostrando uma progressão salarial clara ao longo da carreira.

O trabalho remoto possui forte presença no mercado de dados, reforçando a tendência global de trabalho distribuído na área de tecnologia.
